# 🇮🇳 Project Netr

> **Every Dream Deserves an Opportunity.**

Project Netr is an AI-powered opportunity discovery platform designed to bridge the gap between citizens and the government schemes, scholarships, grants, welfare programs, and career opportunities they are eligible for.

Instead of searching across hundreds of government websites, users simply tell Project Netr who they are, and the platform intelligently discovers opportunities tailored to their profile.

---

## 🌟 Vision

Millions of people miss life-changing opportunities every year simply because they never know they exist.

Project Netr aims to become the **AI-powered gateway to opportunities**, making government support, education, employment, and financial assistance accessible to every citizen.

Our mission is simple:

> **No deserving citizen should miss an opportunity due to lack of awareness.**

# 🚀 Getting Started

Clone the repository:

```bash
git clone https://github.com/Akshat19G/Project-Netr.git
```

Navigate to the project:

```bash
cd Project-Netr
```

Install dependencies:

```bash
npm install
```

Run the development server:

```bash
npm run dev
```

Build for production:

```bash
npm run build
```
